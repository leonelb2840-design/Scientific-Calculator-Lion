# 🦁 Scientific Calculator Lion - Legacy Edition

¡Bienvenido al rugido de la precisión! Esta es una versión rescatada y optimizada de la **Calculadora Científica Leonel**, enfocada en la funcionalidad pura y el rendimiento ligero.

## 🚀 Características del Proyecto
Este repositorio mantiene viva la esencia de la calculadora original, integrada como una **PWA (Progressive Web App)** de alto rendimiento.

* **Interfaz Minimalista:** Diseño limpio para un cálculo rápido y sin distracciones.
* **Modo Offline:** Gracias a su Service Worker "Master", funciona sin necesidad de internet (ideal para el liceo o zonas sin señal).
* **Instalable:** Puedes añadirla a la pantalla de inicio de tu Android como una app nativa.
* **Estructura Ligera:** Código optimizado en HTML, CSS y JavaScript puro.

## 🛠️ Tecnologías Usadas
* **HTML5:** Estructura semántica.
* **CSS3:** Diseño responsivo y moderno.
* **JavaScript (Vanilla):** Lógica de cálculo y motor del sistema.
* **Service Workers:** Para la persistencia de datos y caché offline.

## 📱 Cómo Instalar
1. Abre el enlace del proyecto en tu navegador (Chrome recomendado).
2. Toca los tres puntos arriba a la derecha.
3. Selecciona **"Instalar aplicación"** o **"Agregar a la pantalla de inicio"**.
4. ¡Listo! Ya tienes el poder del león en tu menú de apps.
